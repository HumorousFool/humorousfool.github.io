Introduction

Why Plugins?

When I first started making plugins, I started with a few basic tutorials and then used that knowledge to create my own ideas. Plugins allowed me to explore my own ideas using a simple, extremely well documented, high-level API. This tutorial assumes that you have some basic knowledge of Java, but I think plugins are the perfect way to strengthen your knowledge of Java and object oriented programming. This tutorial will walk you through the very basics of plugin development to send you on a path of learning and also provide some more specific resources at the end.

Requirements

The list requirements for this tutorial is pretty short:

  • Basic knowledge of object oriented programming
  • The ability to run Minecraft
  • Basic knowledge of Minecraft
    • Like I said, pretty short.
